# Versión Panel Led 1bpp

Este proyecto muestra imagenes a 1bpp, las imagenes tienen buena visibilidad pero dependera del contexto, para imagenes complejas y muchos colores no es lo ideal.
Esta versión no tiene control por PWM.
La logica de los modulos es igual a la [versión de 8bpp](https://github.com/Fdiaz718/Panel-Led-8bpp)
